content 관련 메소드

html() 메소드

            선택된 요소의 content 영역을 설정하거나 반환하는 메소드이다.
            setter : $("선택자").html("내용");
                -> 내용에 html 태그가 포함되어 있는 경우 실제 태그로 인식됨.

            getter : $(선택자).html();
                -> 해당 요소의 content로 작성된 내용에서
                    태그까지 포함하여 읽어옴.

text() 메소드

    선택된 요소의 content 영역을 설정하거나 반환하는 메소드이다.
    setter : $("선택자").text("내용");
        -> 내용에 html 태그가 포함되어 있는 경우 무시하고 문자열로 인식함.

    getter : $("선택자").text();
        -> 해당 요소의 content로 작성된 내용에서
        태그를 미포함하여 읽어옴.
