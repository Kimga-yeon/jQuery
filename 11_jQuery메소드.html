each() 메소드

Java에서 사용하는 향상된 for문(for each문)과 비슷한 메소드로
객체나 배열의 요소를 반복 접근할 때 사용. 

            1. $.each(object, function(index,item){ })
                - object : 배열 또는 객체
                - index : 현재 접근중인 배열의 인덱스 또는 객체의 key값을 의미
                - item : 현재 접근 중인 배열 요소의 값 또는 
                [          객체 속성의 value값을 의미

            2. $("선택자").each(function(index, item){})
                - $("선택자")를 이용하여 여러 요소를 한번에 선택
                    -> 배열의 형태로 반환됨
                    -> 반환된 배열을 반복접근함.
